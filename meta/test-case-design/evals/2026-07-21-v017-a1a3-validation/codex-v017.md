  - 验收标准 #1：合法 `.xlsx` 逐行解析并入库，响应给出成功导入行数和被跳过行。
  - 验收标准 #2：导入完成后可在列表查询到本次记录，默认按 Excel 原始行顺序展示。
  - 验收标准 #3：支持按工号、打卡日期区间、部门分页查询，默认每页 20 条。
  - 验收标准 #4：工时由后端按“下班时间 − 上班时间”计算并存储，保留两位小数，不接受调用方传入。
  - 业务规则 §1.1：工号必须为 8 位数字；非法工号所在整行跳过，不阻断其他行，并报告跳过行号。
  - 业务规则 §1.2：相同“工号 + 打卡日期”是同一考勤；重复导入时最后一次覆盖旧值。
  - 业务规则 §1.3：上班时间必须早于下班时间；上班时间晚于下班时间的行跳过并报告。
  - 业务规则 §1.4：打卡日期必须位于所选月份，跨月行跳过并报告。
  - 业务规则 §1.5：Excel 数据行的 5 列依次为工号、打卡日期、上班时间、下班时间、部门；解析包含日期/时间格式转换与工时计算。
  - 业务规则 §1.6：校验不合格行进入 `skipped`，其余合格行继续批量落库；校验项含工号、时间先后、月份边界和必填非空。
  - 异常场景 #1：非 `.xlsx` 文件返回消息“仅支持 .xlsx 格式”，且不入库。
  - 异常场景 #2：Excel 无数据行时返回成功，导入数为 0。
  - 异常场景 #3：数据行缺少打卡日期时，该行跳过并报告，不阻断其他行。
  - 异常场景 #4：服务器内部错误使用业务码 `5000`。
  - 接口契约 #1：导入端点为 `POST /api/attendance/import`，请求为 `multipart/form-data`，文件字段名为 `file`。
  - 接口契约 #2：导入响应信封为 `{ code, message, data: { imported, skipped: [{ row, reason }] } }`；成功码 `0`，格式错误码 `4001`，内部错误码 `5000`。
  - 接口契约 #3：查询端点为 `GET /api/attendance/page`；参数为 `empNo`（可选）、`dateFrom`、`dateTo`、`dept`（可选）、`pageNo`（默认 1）、`pageSize`（默认 20）。
  - 接口契约 #4：查询响应信封为 `{ code, message, data: { total, records } }`；`AttendanceVO` 含 `empNo, workDate, clockIn, clockOut, workHours, dept`。
  - 接口契约 #5：`t_attendance` 以 `(emp_no, work_date)` 唯一；`emp_no` 最长 8 字符、`dept` 最长 64 字符，`clock_in/clock_out/dept` 可空；表中没有原始 Excel 行号列。
  - 接口契约 #6：当前设计的分页查询直接读取 `t_attendance`，没有显式 `ORDER BY`；同批记录的 `created_time` 相同。
  - 接口契约 #7：纯函数 `parseAttendanceRow` 是明确测试缝隙，必须覆盖 5 列映射、工时计算、格式解析，并包含工时异常和非法日期格式失败场景。
  - 接口契约 #8：导入为同步“解析 → 校验 → 批量落库 → 返回结果”流程；查询直接经 MyBatis-Plus 分页读取数据库，设计未描述缓存或异步下游。

### §1 输入契约边界登记

| 输入 | 有效类与边界 | 无效类（每种原因单列） | 对应用例/归宿 |
|---|---|---|---|
| `file` | 合法 `.xlsx`；零数据行 | 非 `.xlsx`；缺失字段；扩展名正确但内容损坏 | TC-F-001、TC-E-014～016、TC-ERR-001 |
| Excel `empNo` | 8 位数字 | 7 位、9 位、含非数字、空值 | TC-E-001～005 |
| Excel `workDate` | 所选月首日、末日 | 月首前一天、月末后一天、空值、非法日期格式 | TC-E-009～012、TC-ERR-002～003；月份输入冲突见澄-3 |
| Excel `clockIn/clockOut` | 相差 1 分钟；可计算小数工时 | 相等、上班晚于下班、任一缺失、格式非法 | TC-E-006～008、TC-E-017～019、TC-ERR-004 |
| Excel `dept` | 普通非空值；64 字符存储边界 | 空值；65 字符 | TC-F-004、TC-E-020～022 |
| 查询 `empNo` | 合法 8 位；省略 | 非 8 位 | TC-F-002、TC-E-023、TC-E-030 |
| 查询 `dateFrom/dateTo` | 一般区间；同日区间 | 逆序、缺项、格式非法 | TC-F-003、TC-E-024～027 |
| 查询 `dept` | 普通非空值；省略 | 空字符串语义未定义 | TC-F-004、TC-E-023、TC-E-020（同一业务字段缺口见澄-8） |
| `pageNo` | 默认 1；显式正整数 | 0/负数；非整数的绑定结果未定义 | TC-F-005～006、TC-E-028；非整数规则见澄-10 |
| `pageSize` | 默认 20；显式正整数 | 0/负数；上限未定义 | TC-F-005～006、TC-E-029；上限规则见澄-10 |
| 多查询条件 | 单条件；省略可选条件 | 多条件组合语义未定义 | TC-F-002～004、TC-E-023、TC-E-031 |
