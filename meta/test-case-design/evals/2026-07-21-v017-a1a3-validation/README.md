### 跨模型差异（有价值的观察）

A3 靶子上 **Claude 双标（冲突 + 可观察性）、Codex 单标（冲突）**，两格最终都不写死。说明可观察性核与冲突门在「源要求的不可测断言」上**互为冗余、并不矛盾**——是好的防御纵深：Claude 证明二者可正交共存，Codex 证明冲突门能独立兜底。

### 效力边界（诚实标注）

- **n=1 合成任务**，跨任务普适性未证。
- **靶子设计偏差**：A3 靶子做成了「源明确要求的顺序」（AC-2），天然带冲突维度 → Codex 直接走冲突门、可观察性核让位。**可观察性核在「源沉默、零冲突」的纯场景仍未被隔离测到**。要纯测需诱导模型写出不可测断言——但好模型会主动回避（Codex 的 `TC-F-005` 就写「不对内部顺序作额外约定」），故这类**出口安全网天然难正向端到端触发**。留待补测或注入式单元验证。
