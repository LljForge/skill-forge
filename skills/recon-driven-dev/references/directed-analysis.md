# ① 定向分析 细则（现状侦察 · 派发路由 + 主会话实测门 + 封存边界）

> ① 阶段的细则,由 SKILL.md ① 指针引入。侦查 sub-agent 的契约(四样 / 软硬分等 / 探查纪律 / 出口自检)单一权威在 `recon-agent.md`,本文不复述;派发与降级的通用机制住 SKILL.md「派发与降级」,本文只补 ① 的实例化。

## 职责与边界

**职责**:动需求/方案前,把这次需求主题落进的那片现有代码摸清楚——有啥、牵动谁、埋了什么雷——按主题摸**全集**、全带锚点,喂给 ②。

**边界**:只摆现状事实(不细化需求、不定改动范围、不出方案、不判好坏)——**含码上读不到的运行时/数据态**,按 `recon-agent.md` 的「软/硬分等」标记与核验。它是给 ② 的**现状参考、不是权威上限**。

输入是用户一句话粗需求,产出一份摸底报告(`directed-report.md`)——它是 ②③ 共用的上游参考。

## 派发侦查 sub-agent

翻代码的体力活(实际去搜"谁调用了这个 Service"、追调用链)**派给一个自实现的侦查 sub-agent**,别在主上下文亲自 grep——避免主对话被一堆中间检索结果污染。

派发与降级按 SKILL.md「派发与降级」范式走;① 的实例化:

- **prompt 体** = `recon-agent.md`;**追加在末尾**的几样:粗需求 + 本次主题 / 落点焦点 + 几条已知坑的方向 + 本次产物目录路径(主 agent 已建好)。
- **工具面**:`Read` / `Grep` / `Glob` + `Write`(源码只读;`Write` 仅写 `directed-report.md`;约束见 `recon-agent.md` 入口)。
- **降级态**:主会话亲自摸、没有隔离收益,须把 `recon-agent.md` 的契约当外部尺逐条核,把检索噪声尽量挡在叙事外。

四样契约 / 软硬分等 / 探查纪律 / 出口自检都住 `recon-agent.md`、本文不复述;报告契约与下面这道质量门**与跑在哪一级无关**,派发只是执行方式。

## ⏸ 主会话实测门(① 的质量门 · 防空转的真牙)

定向分析很薄,半成品报告喂进 ② 就让后面全建在沙上。侦查 agent 出报告前已照 `recon-agent.md` 的**出口自检**自核过;**这道门在主会话当场再守、不拖到 ③**——收到报告后逐项核对:

- **照 `recon-agent.md` 出口自检逐项复核**(四样齐全 / 锚点真实 / 逐载体踩住覆盖地板 / 坑是真·非显性料)——清单住 `recon-agent.md`、此处不复述;抽查几个锚点跳过去看、对得上当前代码。
- **支撑坑的软推断核验了吗?** 挑**支撑坑、尤其反直觉陷阱 / 协同护栏两味**的 `[未核验]` 条——核验目标就是那条里那个读不到的取值/走向:**先扫本会话可用工具面(含 MCP)**,命中**现成只读、零副作用**的手段(不为此搭环境 / 写新测试)就把真实值取出来比对,一致 → 升 `[实测]`、不一致 → 主会话就地改正(报告未封存、直接修);够不着就坦诚留 `[未核验]` 放行。报告自标「可能 / 非目标 / 无法确认」的边缘条留标记即可。

> 实测是**主会话(主 agent)这道门**的活、不指望只读码的侦查 agent——它只负责标 `[未核验]`。这正是 ① 指针点名"主会话实测门、非泛质量门"的所指:不是只复核"四样齐全"就放行。

**不过门** → 打回 recon-agent 重摸(缺行 / 个别锚点这类**定点打回补摸即可、不必整报告重钻**;降级态则主会话就地补),**绝不让半成品报告进 ②**。

## 封存边界（① 状态流 · 单一权威）

① 的状态流:**recon-draft → main-gate-passed → user-approved → sealed**(design §8.1)。

- 主会话实测门通过 = `main-gate-passed`,此时**先把报告交用户审阅、尚未封存**。
- 用户要求修改 → 回 `recon-draft`,改后**重过实测门**(不是直接改完就进②)。
- 只有**用户批准后**(`user-approved`)才把 `directed-report.md` 封存(`sealed`)。
- 封存前(含 `main-gate-passed` 待用户审阅期)它还是草稿、就地直接改、不计订正。
- 封存后任何环节用证据推翻它写下的一条事实主张,走「事实订正」——**全程订正总账(各阶段落点 + 范式)见 SKILL.md「事实订正」,本文不持全程落点**。
